用到的 linux 命令记录

Jun 5, 2018

linux 命令比较多，很多经常不用会忘记。

jobs 相关

  • command & 让进程在后台运行
  • ctrl + z 当前命令后台，并暂停
  • jobs 查看后台运行的进程
  • fg %n 让后台运行的进程n前台运行 (n 为 jobs 中的进程编号)
  • bg %n 进程n后台运行 (n 为 jobs 中的进程编号)
