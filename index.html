  • 搭建 Hexo 博客遇到的问题

    Jun 4, 2018

    Hexo 安装和上手比较简单，但是在使用过程中还是遇到了一些问题，大部分可以通过查看文档解决。

    遇到的问题

    添加标签

    我使用的主题是没有 tags 的，所以要自己添加 tags，然后在添加过程中遇到不少问题。

    • 通过 hexo new page tags 创建新的页面，source 目录下对应生成 tags/index.md 文件。
    • 在使用的主题的_config.yml中添加对应的 url 。
    • 在当前主题的 layout 目录创建 tags.jade 文件 。

    • 在 layout 添加 tags 对应的页面（显示所有的 tags），在这一步，出现了不少问题。

      • 首先是 page 如何和 layout 关联，这个主要是通过设在 tags/index.md 中指定对应的 layout （tags）。

      • 另外一点就是 hexo 模版之间的对应关系 。

          • 所以 某一个具体的 tag 和 archives 是共用一个 archive.jade 文件， 因此可以通过 page.category 、page.tag 、page.archive 来判断当前的page 并做相应的处理。
